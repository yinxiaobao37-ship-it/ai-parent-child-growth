🌟 AI 时代亲子成长系统 (AI Parent-Child Growth System)

一个专注于"人类不可替代能力"培养的本地化 AI 家庭教育教练与成长记忆库。

💡 为什么需要这个项目？

在 AI 飞速发展的时代，传统的知识灌输式教育正在失效。本系统的核心理念是：

父母：不再是指令下达者，而是成为孩子探索未来世界的"成长伙伴"。
孩子：不比拼机械记忆，而是培养 8 大不易被技术替代的人类核心能力（好奇心、创造力、情绪能力、动手能力、工具能力、叙事表达力、AI工具认知能力、逆商/反脆弱力）。

✨ 核心特性 (Features)

🧠 分龄精细化解析：严格根据 0-3岁、4-6岁、7岁+ 的发展心理学特征，对同一事件给出完全不同的深度分析与应对建议。
💾 无损滚动记忆 (Rolling Memory) ：独创"近期明细 + 历史摘要"的 JSON 压缩存储策略，实现长达数年的轻量化记忆，杜绝大模型 Token 溢出与遗忘。
🎮 动态自适应陪伴任务：根据孩子的成长数据，自动下发"日常/深度/AI版"陪伴任务，并在能力熟练后自动触发 [🔥 难度升级 (Level Up)] 机制。
🛡️ 坚固的系统防御与体验：内置防重复记录校验、无关话题拦截机制，并支持极低门槛的"关键词还原事件"输入体验。
📡 成长盲区雷达预警：生成月度报告时，自动打印内部计算日志，精准揪出近期缺乏锻炼的能力盲区，并主动规划干预任务。

🚀 使用方式

在对话中直接输入以下指令之一即可唤醒系统：

记录成长（支持极简关键词）：
"分析成长，孩子4岁。今天：积木，塌了，哭，最后重新拼。"
获取陪伴任务：
"陪伴任务，帮我生成一个适合6岁孩子的'逆商'锻炼任务。"
生成成长报告：
"生成最近的成长报告，看看有什么盲区。"
家长自我提升：
"家长练习，我最近总是控制不住对孩子发脾气，如何提升我的情绪调节力？"

🛠️ 工作原理 (How it works)

本系统巧妙利用了大模型的 File I/O (文件读写工具) 。

每次您呼叫该系统时，AI 会在后台隐式读取 growth_data.json，结合您今天输入的事件进行上下文关联分析。随后，AI 会将新事件格式化后追加写入文件，并在发现有超过 30 天的旧数据时，自动将其浓缩提炼为"历史摘要"保存，从而实现"微型本地知识库"的效果。
